# DART 금융 대시보드 웹 앱 실행 가이드

요청하신 금융 데이터 시각화 웹 애플리케이션(`app.py`) 구현이 완료되었습니다.
Streamlit을 사용하여 구축되었으며, 아래 명령어로 실행할 수 있습니다.

## 실행 방법

1. **터미널**을 엽니다 (PowerShell 또는 CMD).
2. 프로젝트 폴더(`c:\Users\User\Documents\stock`)로 이동합니다.
3. 다음 명령어를 입력합니다.

```powershell
.\run_app.bat
```
(또는 탐색기에서 `run_app.bat` 파일을 더블 클릭하세요.)

만약 직접 명령어를 치고 싶으시다면, 아래와 같이 전체 경로를 사용해야 할 수 있습니다:
`"C:\Users\User\AppData\Local\Programs\Python\Python312\python.exe" -m streamlit run app.py`

4. 명령어를 실행하면 자동으로 브라우저가 열리며 대시보드가 나타납니다.
   (자동으로 열리지 않으면 터미널에 표시된 `http://localhost:8501` 주소를 주소창에 입력하세요.)

## 주요 기능

- **종목 검색**: 사이드바에서 원하는 종목명(예: 삼성전자, SK하이닉스)을 입력하면 데이터가 갱신됩니다.
- **주가 차트**: FinanceDataReader를 통해 최근 주가 추이를 Candlestick 차트로 보여줍니다.
- **재무 분석**: DART API를 통해 매출액, 영업이익, 순이익을 가져와 분기별 실적으로 가공하여 보여줍니다.
- **시각화**: 비용 구조(Pie Chart)와 이익률 추이(Bar+Line Chart)를 제공합니다.

## 주의사항

- **데이터 로딩 시간**: DART API에서 데이터를 연도별/분기별로 수집하므로 처음 로딩 시 시간이 다소 걸릴 수 있습니다. (진행률 바가 상단에 표시됩니다.)
- **종목 검색**: 상장 종목이 아니거나 DART에 등록된 이름과 다를 경우 데이터를 찾지 못할 수 있습니다.
